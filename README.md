# Soup2Bowl

A simple, responsive catering website for **Soup2Bowl** — handcrafted soup catering for any occasion, by Shelley Wirt.

## Pages

| File | Description |
|---|---|
| `index.html` | Main site — hero, menu, catering packages, about, and contact/order form |
| `style.css` | All site styling (responsive, mobile-friendly) |

## Features

- **Hero section** — tagline and call-to-action buttons
- **Signature soup menu** — 6 soups with descriptions and per-serving prices
- **Catering packages** — Starter, Classic, and Banquet tiers
- **About section** — origin story
- **Order / contact form** — name, email, phone, guests, date, package selection, and notes

## Running Locally

Open `index.html` directly in any modern web browser — no build step required.
